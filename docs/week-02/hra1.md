# Week 2 - HRa1 - Quantitative analyses and Type I & II errors

## What you will learn

- Refresh your knowledge on how to formulate hypotheses (H<sub>0</sub> and H<sub>a</sub>/H<sub>1</sub>).
- Define a Type I and a Type II error.
- Understand how a Type I and Type II error is made.
- Evaluate the potential costs of a Type I and Type II error.
